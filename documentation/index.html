Getting Started

Welcome to Waxon - VueJS Personal Portfolio Template We would like to thank you for choosing our template - Waxon.

It’s built with VueJS. It’s fully responsive and looks well. It is a modern design and high-speed performance VueJS template. You can use it for Personal Portfolio Landing, Minimal Portfolio, Freelancer Portfolio, Designer Portfolio.

Requirements

There are system requirements in order to install and setup this template and its components properly. Make sure that you are running the node.js & npm install.

Recommended node and npm 

                        
                            node install
                            npm install

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies. 

                        
                            npm install

Development server

Run npm run serve for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files. 

                        
                            npm run serve
